# AI Sentiment Analysis & Mental Health Dashboard

An interactive dashboard built with **Streamlit** to analyze sentiment from various e-commerce platform reviews and real-time user input.

## Features
- **Real-time Analysis:** Analyzes user-provided text for sentiment (Positive, Negative, Neutral).
- **Multi-Platform Insights:** Visualizes sentiment data from Daraz, Amazon, Walmart, etc.
- **Multilingual NLP:** Specially tuned to handle Singlish (Romanized Sinhala) and English.

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run Dashboard.py`
